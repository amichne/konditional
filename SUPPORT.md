# Support

## Getting help

For questions about usage, design, or migration:

- Search existing GitHub Issues first.
- Open a new issue with context, expected behavior, and current behavior.

Helpful details to include:

- Kotlin and JDK version
- Operating system
- Module(s) in use (for example, `konditional-core`, `konditional-runtime`)
- Minimal reproducible snippet or test

## Bug reports

For defects, use the bug issue template in this repository.

## Security issues

For potential vulnerabilities, follow `SECURITY.md` and report privately.
