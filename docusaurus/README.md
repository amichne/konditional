# Docusaurus docs (parallel to MkDocs)

This repository keeps MkDocs as-is and adds a minimal Docusaurus setup in `docusaurus/`.

## Local dev

```bash
cd docusaurus
npm install
npm run start
```

If you keep `baseUrl: '/konditional/'`, the dev server will be under `http://localhost:3000/konditional/`.

## Build

```bash
cd docusaurus
npm install
npm run build
```
