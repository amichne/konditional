# Konditional Spec Reference

## Essential types

- `SurfaceRouteCatalog` - canonical route + operation definitions
- `SurfaceDtos` - snapshot/mutation/error/codec DTOs
- `SurfaceSchemaRegistry` - OpenAPI component schema registration
- `SurfaceOpenApiSpecBuilder` - document construction + deterministic ordering
- `SurfaceOpenApiSpecGenerator` - build entrypoint used by Gradle task
